# Bizconnect Technologies

Premium React webapp for Bizconnect Technologies — digital presence and technology solutions for businesses across Ghana (websites, social media marketing, and more).

## Stack

- **React 18** + **TypeScript**
- **Vite** (build tool)
- **Framer Motion** (animations)

## Commands

```bash
npm install   # install dependencies
npm run dev   # start dev server (http://localhost:5173)
npm run build # production build
npm run preview # preview production build
```

## Responsive breakpoints

- **Desktop**: default layout (multi-column)
- **Tablet**: ~960px — single column sections, 2-col steps/footer
- **Mobile**: ~640px — hamburger menu, stacked CTAs, single column everywhere

## Design

- Dark emerald theme with subtle gradients and noise texture
- Outfit (headings) + DM Sans (body)
- Scroll-triggered and hover animations via Framer Motion
- Reduced-motion respected in `index.css`
